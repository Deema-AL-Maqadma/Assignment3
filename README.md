# 🌐 Web Programming 1 – Assignment 3 (Login Page)

This repository contains the third assignment completed in the **Web Programming 1** course.  
It demonstrates how to build a styled **Login Page** using HTML and CSS.  
The page includes input fields, labels, a login button, and a link for password recovery.

---

## 📁 Folder Structure

| File Name        | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| LoginPage.html   | Assignment creating a login page with username and password fields          |
| Style.css        | CSS file providing styling for the login page (layout, colors, fonts, etc.) |

---

## 🧠 Topics Covered

- HTML form inputs (`<input type="text">`, `<input type="password">`)
- Labels for accessibility (`<label>`)
- Buttons (`<button type="submit">`)
- Links (`<a href="...">`) for password recovery
- CSS styling for layout and design
- Flexbox centering (`display: flex; justify-content; align-items`)
- Box styling: padding, border-radius, shadows
- Typography and color customization
